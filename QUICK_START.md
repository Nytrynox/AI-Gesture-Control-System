## 🎯 How to Use

1. Launch the application (see Quick Launch above)
2. Click **"▶ Start Camera"**
3. Position your hand 1-2 feet from camera
4. Perform gestures from GESTURE_CONTROLS.md
5. Adjust sensitivity slider if needed (default: 0.7)
6. Enjoy complete laptop control! 🎉

---

## 🛠️ Settings

### Recommended Settings:
- **Sensitivity**: 0.7 (balanced)
- **Smoothing**: 0.3 (responsive)
- **Mouse Control**: ON
- **System Actions**: ON

### If Gestures Not Detecting:
- Lower sensitivity to 0.5-0.6
- Improve lighting
- Ensure hand fully visible
- Hold gesture for 0.5 seconds

## 🐛 Troubleshooting

### Camera Not Working
```bash
# Check camera permissions in System Preferences > Security & Privacy > Camera
```

### Low Performance
- Close other camera apps
- Reduce number of open applications
- Check CPU usage in Activity Monitor

### Gestures Not Working
- Check sensitivity setting (try 0.5-0.6)
- Verify toggles are enabled
- See gesture_control.log for errors
